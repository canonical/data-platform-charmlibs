# Python libraries for Data Platform

This monorepo hosts the source code for Canonical's Data Platform charm libraries.

To get started with contributing, read [CONTRIBUTING.md](./CONTRIBUTING.md).

### 1. Interfaces

#### Purpose

This is Data Interfaces V1, as a pip package.
It will be used for charmlibs, but also for libraries that require some parts of data interfaces v1, but could not use it because it was in a charm-lib.

How to use: see the module documentation.
